Quest Project

The Quest Project has been called to life for the JOTA BusjesRally of the dutch boy- & girlscout region Vlietstreek and surroundings. The Quest Projects aim to provide an open, cross-platform, full-fledged solution to create and publicize Quests, which can be explored and executed using an "App"-like platform.

Components

Quest Project consists of various components to make-up and support the App

Quest JSClient

The JSClient is a HTML based client, which can be run from a browser, but is also used for the App. It contains the entire client for the Quest Server, and power the entire Quest App.

A live demo can found at http://app.quest.jeffenmau.org/.

Quest App

The Quest App is an Apache Cordova implementation of the JSClient which provides an installable app. This is the sourcecode used to build the production org.jeffenmau.quest.app Application.

The Android version of the App can be found in Google's Play Store.

Quest Server

The Quest Server is a Symfony2 Project containing the Frontend, Backend and API required to host your own quests. The JSClient / App connect to this server to get the Quests' Questions and to submit Answers.

A live demo can found at http://server.quest.jeffenmau.org/.
